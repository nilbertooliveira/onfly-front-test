## Ambiente Docker

Arquitetura baseada em clean, SOLID, DRY, calisthenics, DDD, KISS e YAGNI.

### Passo a passo

1. Clone o repositório:
   ```
   git clone https://github.com/nilbertooliveira/onfly-front-test.git
   ```

2. Faça o build e suba os serviços:
   ```
   docker compose up -d
   ```
